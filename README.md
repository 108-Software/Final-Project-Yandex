## Описание
Проект представляет собой удобный инструмент планирования задач с современным WEB-интерфейсом. Система позволяет эффективно управлять личными и рабочими задачами с поддержкой гибкого планирования повторений.

### Умное планирование повторений
- **Ежедневные задачи** - выполнение с заданным интервалом в днях (1-400 дней)
- **Еженедельные задачи** - выполнение в указанные дни недели (1-понедельник, 7-воскресенье)  
- **Ежемесячные задачи** - выполнение в конкретные числа месяца (1-31) или дни с конца месяца
- **Ежегодные задачи** - автоматическое повторение в ту же дату каждый год


## Примеры правил повторения

| Правило | Описание |
|---------|-----------|
| `"d 7"` | Каждые 7 дней |
| `"y"` | Каждый год в эту же дату |
| `"w 1,3,5"` | По понедельникам, средам и пятницам |
| `"m 15"` | Каждое 15 число месяца |
| `"m -1"` | В последний день месяца |
| `"m 10,20 3,6,9,12"` | 10 и 20 числа в марте, июне, сентябре и декабре |
